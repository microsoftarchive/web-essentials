Animation

Get started

To get started you only need to add two classes. First add the animated class. Second class is the actual animation, to find what will work for you simply test and pick one from the list above. My favorites animation families are slideIn, fadeIn, bounceIn.

In the example below I'm fading in the Wunderlist logo

<div class="animated fadeIn"></div>

Infinite

Sometimes you want an animation to continue forever. So just add infinite and it'll never stop.

<div class="animated zoomOutDown infinite"></div>

Delay

If you want to add a delay, simply add the class delay

<div class="animated zoomIn delay"></div>

Durations

We have three durations at the moment. Fast (0.5s), Normal (Default, 1s), Slow (2s).

<div class="animated bounceInDown slow"></div>
<div class="animated bounceInDown normal"></div>
<div class="animated bounceInDown fast"></div>
