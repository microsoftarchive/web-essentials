01. Typography

Typography is very important. It’s the only element that speaks to both sides of the brain. It has to be both beautiful and legible. Typograpy is what converys our message and outputs our users content. As the UI of Wunderlsit continues to take the backseat, we have to call on other elements like typography to help carry on our character. Lato does this very well. It has a playful but structured nature. It has a great x-height is makes reading a breeze and looks great on all our platforms.

04. Whitespace

Shorthand Description
m margin
p padding
t top
r right
b bottom
l left
x x-axis (left and right)
y y-axis (top and bottom)
n negative (margin only)
0 0
half --space-half = .25rem
1 --space-1 = .5rem
2 --space-2 = 1rem
3 --space-3 = 2rem
4 --space-4 = 4rem

Margin and padding utilities are based on a global white space scale defined with variables. These utilities use a shorthand naming convention.

Margins

Change or reset default margins using the global white space scale. Negative x-axis margins are used to offset padding. Margin auto is used to horizontally center block-level elements with a set width.

Good morning! .m0

Hallo .mb2

How are you? .bold

<div class="outline-blue clearfix">
  <h1 class="m0">Good morning! <span>.m0</span></h1>
  <p class="mb2">Hallo <span>.mb2</span></p>
  <p class="bold">How are you? <span>.bold</span></p>
</div>

Padding

Padding utilities are only available in symmetrical orientations. This is to normalize the spacing used around elements.

Good morning! .p1

Hallo .px4

How are you? .px4 > .mxn2 

<div class="outline-blue clearfix">
  <h1 class="p1">Good morning! <span>.p1</span></h1>
  <p class="px4">Hallo <span>.px4</span></p>
  <p class="px4"><span class="block">
    <span class="mxn2">
      How are you? <span>.px4 &gt; .mxn2</span>
    </span>
  </span></p>
</div>

Margins and padding should never be declared outside of these utilities. This is meant to help create consistent rhythm and avoid magic numbers. If, for some reason, the default white space scale does not accomodate your design, customize and extend it before implementation.

Full height section

To make a full height section, the html and body tags must first be full height. A valid example html document is: 

<!doctype html>
<html class="body-full-height">
  <head>
    <title>Full height section example</title>
    <link rel="stylesheet" href="css/base.min.css">
  </head>
  <body class="body-full-height">
    <div class="full-height white bg-blue">
      <div class="center py4">
        <h1>First</h1>
      </div>
    </div>
    <div class="full-height white bg-green">
      <div class="center py4">
        <h1>Second</h1>
      </div>
    </div>
  </body>
</html>

This example can be seen at full-height-example.html

Also, the first section of this page uses this technique. It also employs a table layout for vertical centering.

06. Grid

Clearfix

The grid allows a lot of flexibility in creating complex layouts. Start by using a .clearfix container. 

<div class="clearfix">
</div>

Simple columns

Add columns using the .col and grid width .col-N classes. .col floats elements left and sets box-sizing to border-box. .col-N sets width according to a 12 column grid. The total number of columns in a row should add up to 12.

.col.col-6
.col.col-6
<div class="clearfix outline-blue mb2">
  <div class="col col-6">.col.col-6</div>
  <div class="col col-6">.col.col-6</div>
</div>

Gutters

Use padding and negative margin utilities to create gutters based on the white space scale. When using negative margin, be sure to compensate for the extra width created with a padded parent element or by using overflow hidden. Otherwise, horizontal scrolling may occur.

.col .col-5 .px2
.col .col-7 .px2
<div class="clearfix outline-blue mb2">
  <div class="col col-5 px2"><div>.col .col-5 .px2</div></div>
  <div class="col col-7 px2"><div>.col .col-7 .px2</div></div>
</div>

Negative margins

An example using negative margin to expand the columns in an already padded section, which appears to have a single gutter in the center.

I am the normal size.
.col .col-6 .px4
.col .col-6 .px4
<div class="px4 py2 outline-blue bg-off-white mb2">
  <div class="mb1">I am the normal size.</div>
  <div class="clearfix mxn4">
    <div class="col col-6 px4"><div>.col .col-6 .px4</div></div>
    <div class="col col-6 px4"><div>.col .col-6 .px4</div></div>
  </div>
</div>
