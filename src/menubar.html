How to build the menubar

The basic shapes are to have one left, one right, and then one fluid middle. Here is a simplified example:

1. Floats

<% code(insert: true) do %>
.left
.right
.overflow-hidden
<% end %> <% code(insert: true) do %>
logo
wordmark
bell
bubble
avatar
chevron
.overflow-hidden
<% end %>

3. Make responsive

The responsive class .md-show means "only show when medium size screen or above." Go ahead and resize the window.

<% code(insert: true) do %>
logo
wordmark .md-show
bell .md-show
bubble .md-show
avatar
chevron
.overflow-hidden
<% end %>
